# Product UI kit — Analyze

Hi-fi recreation of the Polar in-product dashboard (Analyze surface). Source: `uploads/hero-02.jpg` only — no in-product codebase was provided.

## Components
- `aside` — 64px rail with stacked icons; active pill + alerts-center accent (yellow)
- `.topbar` — time selector, views, search with ⌘K, date picker
- `.card` — light mode card with header + body
- `.kpi` tiles — indigo dot, delta pill, "Explore metric" ghost button
- `.chart` — bar + point overlay chart, minimal gridless
- `.tab` — pill segmented control
- `.toggle` — pill switch
