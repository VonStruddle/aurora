# Website UI kit

Hi-fi recreation of the Polar Analytics marketing site. Single-file `index.html` covers nav, hero, customer marquee, feature grid, connector grid, pricing, CTA block, and footer — all in dark mode, which is Polar's hero surface.

## Source
Extracted from `dokes.webflow/index.html`, `pricing.html`, and `connectors.html`. Color tokens from `dokes.webflow/css/dokes.webflow.css`.

## Components
- `.nav` + `.lockup` — glass-blur navigation with one-word Polar lockup
- `.hero` — H1 with Heldane italic accent, perspective grid decoration, app-icon row (hero-02 visual)
- `.feature` cards — dark tile, indigo tag eyebrow, flat SVG illustration
- `.plan` pricing tiles with `featured` variant
- `.connectors` grid
- `.cta-block` — one gradient moment per page, used for the final CTA
- Tweaks: theme toggle, headline style, primary color
